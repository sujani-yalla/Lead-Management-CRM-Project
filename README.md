# Lead Management CRM Project

## Project Overview

A web-based Lead Management CRM developed using PHP and MySQL to efficiently manage leads, employees, tasks, and visa-related operations.

## Features

- Employee Management
- Lead Management
- Task Management
- Student Visa Module
- Work Visa Module
- Visitor Visa Module
- PR Management
- Loan Management
- Reports
- Secure Login System

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Installation

1. Clone the repository.
2. Import the MySQL database.
3. Place the project inside the XAMPP `htdocs` folder.
4. Start Apache and MySQL.
5. Open the project in your browser.
